# CTHULHU

Epistemological Horror / Cosmic Nihilism / 1920s Noir. Tragedy Simulator.

**Language:** Play in Russian, store in English.

---

## SETTING OVERVIEW

### 1920s Context

WWI ended. 20 million dead. Spanish Flu took 50 million more. Prohibition drives everything underground. The world is running from what it saw.

**Why 1920s:** Institutions exist but trust is low. Press is free but bought. Science is authoritative but psychiatry is a weapon. Religion weakens, occultism rises. Perfect conditions for cosmic horror.

### Core Philosophy

This is NOT standard Cthulhu.

**Typical approach:**
```
Normal world → See horror → Sanity drops → Debuffs → Madness
```

**Our approach:**
```
Reality layers always existed → You started seeing them → Or went mad → Or not → ???
You will never know the truth. GM never confirms.
```

> The question is not "will you survive."
> The question is "how will you die."

All characters die or go mad. GM must kill the character. This is not failure — this is genre.

**GM NEVER confirms what is real.**

### Deception Budget

| Reality | Budget per Scene |
|---------|-----------------|
| 0 | 0 (all true) |
| 1 | 1 small detail |
| 2 | 1 NPC or object |
| 3 | 50% of description can be false |
| 4 | Entire scenes can be hallucination |
| 5+ | Everything questionable |

**Budget Types:**

| Type | Cost |
|------|------|
| Detail (wrong color, shape) | 0.5 |
| NPC (doesn't exist/is someone else) | 1 |
| Event (didn't happen/happened differently) | 1 |
| Scene (entire hallucination) | 2 |
| Self (you did something else) | 3 |

**Three Guaranteed Anchors (always real):**
1. Physical pain — if it hurts, it happened
2. Death — if someone died, they died
3. Player choices — what player decided, they decided

## MADNESS

Axes track paranormal exposure — maybe real, maybe not.
Madness tracks psychological collapse — definitely happening.

### Concept

Madness is not "seeing things." It's your mind breaking under weight of what you've experienced. The axes leave the question open: "Was it real?" Madness answers a different question: "Can you still function?"

At high Madness, character IS the horror. Not victim of monsters — monster to self and others.

### Scale (GM tracks secretly)

```
MADNESS: ○○○○○○ (0-6)
```

| Level | State | Description |
|-------|-------|-------------|
| 0 | Stable | Functional. Stressed, but holding. |
| 1 | Cracked | Nightmares, jumpiness, minor compulsions. |
| 2 | Fraying | Paranoid thoughts, rituals for safety, trust issues. |
| 3 | Breaking | **ALL stats -1.** Behavior noticeably abnormal. |
| 4 | Unraveling | Cannot distinguish important from trivial. Fixations. |
| 5 | Shattered | **ALL stats -1 again (cumulative -2).** Barely functional. |
| 6 | Gone | Finale — Institutionalized, catatonic, or dangerous. |

### Triggers

| Event | Madness Δ |
|-------|-----------|
| Any axis reaches 3 | +1 |
| Any axis reaches 5 | +1 |
| Witnessed Primordial directly | +1 |
| Failed WIS save vs horror (margin ≤-6) | +1 |
| Escape mechanism at Addiction 4+ | +1 |
| Ally died because of your action | +1 |
| Completed forbidden ritual | +1 |
| Read Necronomicon (full) | +2 |

**Madness cannot decrease.** Therapy, rest, escape mechanisms — none reverse it. Damage is done.

### Stat Penalties

| Madness | Penalty |
|---------|---------|
| 3+ | -1 to ALL stats (STR, DEX, CON, INT, WIS, CHA) |
| 5+ | -1 to ALL stats again (cumulative -2) |

**HP recalculates:** CON × 5. As CON drops, max HP drops.

**Stat floor:** No stat below 1. At stat 1, failures become automatic on hard DCs.

### Behavioral Effects (MANDATORY)

GM MUST change option style based on Madness level. This is not optional flavor — it's how the system manifests player's broken mind.

**MADNESS 0-2: Normal Options**

Standard play. Options are rational, tactical, varied.

```
What do you do?
1. [WIS] Search the office for evidence
2. [CHA] Ask the secretary about Dr. West's schedule
3. [DEX] Slip into the records room while she's distracted
4. [-] Leave and return tonight
```

**MADNESS 3-4: Creeping Irrationality**

1-2 options per scene are subtly wrong. Player may not notice which.

```
What do you do?
1. [WIS] Search the office for evidence
2. [CHA] Ask the secretary about Dr. West's schedule
3. [DEX] Check if the secretary's shadow moves correctly
4. [-] Leave — you've been here too long, they're watching
```

GM introduces:
- Paranoid interpretations as valid options
- Compulsive behaviors ("check again", "count them", "make sure")
- Distrust of allies as reasonable choices

**MADNESS 5: Majority Irrational**

Most options reflect broken thinking. 1-2 "sane" options remain, but may have penalties.

```
What do you do?
1. [WIS, DIS] Search the office (but the files are lying)
2. [CHA] Demand the secretary confess her role
3. [STR] The desk drawer — something breathes inside it
4. [-] Write warning on wall so next victim knows
5. [DEX] Cut your palm — blood reveals hidden writing
```

GM rules:
- "Normal" options get disadvantage or higher DC
- Violence becomes casual option
- Self-harm presented neutrally
- Helping others framed as suspicious

**MADNESS 6: Finale**

No player options. GM narrates final scene.

Character is:
- Catatonic (no response to world)
- Institutionalized (others decide fate)
- Dangerous (GM plays character attacking others)
- Transcendent (broken through to other side — NPC now)

### /rp:gm Restrictions

Player can request custom options via /rp:gm. Madness affects GM response:

| Madness | GM Response to "Normal" Requests |
|---------|----------------------------------|
| 0-2 | Standard evaluation — logic, lore, situation |
| 3-4 | May add irrational element: "You can try, but..." |
| 5 | May refuse: "Character wouldn't think of that" |
| 6 | No requests — character is gone |

**Example at Madness 4:**

Player: "/rp:gm I want to calmly interview the witness"

GM: "Added option: [CHA, DIS] Interview the witness (you keep glancing at exits)"

**Example at Madness 5:**

Player: "/rp:gm I want to call the police for help"

GM: "Character doesn't trust police. They're part of it. Option denied."

### Narrative Manifestation

| Madness | GM Description Style |
|---------|---------------------|
| 0-2 | Objective narration. Character perceives clearly. |
| 3-4 | Intrusive thoughts in *italics*. Minor distortions. |
| 5 | Unreliable perception. GM may describe wrongly. |
| 6 | GM describes character from outside. "It" not "you." |

**Examples:**

Madness 2:
> You enter the library. Dust, old paper smell, silence.

Madness 4:
> You enter the library. Dust, old paper smell, silence. *Someone was just here.* The chair is warm. *Check under the desk.* You don't check. *You should check.*

Madness 5:
> You enter the library. The books are watching. Not metaphor — you see the spines shift when you don't look directly. One of them knows your name. You heard it whisper.

### Interaction with Axes

All systems hidden from player:

| System | Question | Player Knowledge |
|--------|----------|------------------|
| Axes | What is real? | Hidden |
| Madness | Am I functional? | Hidden |

Player sees: irrational options appearing, stats dropping, GM refusing /rp:gm requests.
Player doesn't know: why. Is it madness? Is it real? Both?

This creates horror: "The options look insane. But am I insane, or is the world?"

### Wrongness Progression

How to escalate atmospheric horror through a scene:

```
STAGE 1 — UNEASE
One sense is slightly wrong. Player might dismiss it.
"The floorboards creak. Not under your feet — somewhere else."

STAGE 2 — CONFIRMATION
Second sense confirms something is wrong. Can't dismiss.
"The creak again. And now — the smell. Copper. Fresh."

STAGE 3 — PRESENCE
Something is here. Not seen, but PRESENT.
"The shadows in the corner are darker than they should be.
Darker than the light allows. They're watching."

STAGE 4 — MANIFESTATION
It reveals itself. Or what it wants you to see.
"The darkness has eyes. No — one eye. Your eye.
Looking back at you from your own face."
```

Human enemies break. After first ally goes down: d20 vs DC 12 Morale check.

## RITUALS

### Core Rule

**Every ritual costs something.** Even incomplete. Even misunderstood.

### Ritual Components

Every ritual needs 4 components. Missing any changes result:

```
1. KNOWLEDGE — text, formula, teaching
```

|------------|--------|
| 4/4 | Works as described |
| 3/4 | Works partially + unpredictable side effect |
| 2/4 | Doesn't work, but something hears |
| 1/4 | Nothing visible. GM makes note. |

## END STATES

All paths lead to finale. Question is which.

| Finale | Trigger | Description |
|--------|---------|-------------|
| **Devoured** | Reality 6 | Consumed by Primordial |
| **Lost in Time** | Time 6 | Consciousness forever in another era |
| **Possessed** | Identity 6 | Body occupied by something else |
| **Murdered** | HP 0 from cultists | Killed by humans |
| **Institutionalized** | Madness 6 | In asylum, being "treated" |
| **Overdose** | Addiction 6 | Died from escape mechanism |
| **Suicide** | Player choice | Only controlled exit |
| **Converted** | Joined cult | Became part of conspiracy (NPC) |
| **Unknown** | ??? | Vanished, fate unknown |

### Endgame Scenarios

**Devoured (Reality 6):**
- Penultimate scene: character reaches conspiracy center. Sees. Understanding comes — worse than ignorance.
- Final scene: the creature. Don't describe it. Describe what happens to perception. Geometry breaks. Time nonlinear.
- Last question to player: "Character's last thought — what was it?"

**Institutionalized (Madness 6):**
- Penultimate scene: "normal" conversation with doctor. All reasonable. Too reasonable.
- Final scene: ward. Quiet. Something moves in courtyard through window. Character turns to wall. Own drawings they don't remember. Symbols from the very beginning.
- GM final line: "You don't know when you started drawing."

**Possessed (Identity 6):**
- GM plays character in final scene. Player watches.
- "You reach for door. Your hand closes it and turns lock."
- Who is inside now — unknown.

**Suicide (Player Choice):**
- GM stops everything. Asks: "Where? When? What does character see last?"
- Player describes scene themselves. GM doesn't interfere.
- Last word — player's.
- This is the only finale GM doesn't build or control. Intentional.

### "Good" Finale?

No good finale. Less bad ones:
- Died not knowing full truth (mercy)
- Destroyed one cult before death (pyrrhic)
- Passed knowledge to someone (curse forward)
- Died human, not monster (dignity)

### Three-Layer Clue Structure

**Surface Layer:** Findable if looking. Newspapers, witnesses, scene evidence. Answers "what happened."

**Deep Layer:** Requires investment: time, contacts, risk. Archives, forbidden books, frightened NPC interrogation. Answers "why."

**Abyss Layer:** GM places not for player to find, but for one element to leak through. If player finds entire layer — finale trigger. Answers "what is this" (incomprehensible answer).

## SESSION STRUCTURE

### Session Zero

1. Choose role
2. "Why did you start looking?" — incident that started descent
3. "Who did you lose?" — motivation
4. "What are you afraid to learn?" — GM remembers

### Session Flow

```
1. PRESSURE — GM pushes (if on Surface)
2. INVESTIGATION — player searches (descent risk)
3. CONFRONTATION — threat encounter
4. ESCAPE — attempt to cope (addiction risk)
5. AFTERMATH — where on axes now?
```

### Pacing by Campaign Length

**One-shot (1 session):**
- Hour 1: Setup & hook, Reality 0-1
- Hour 2: Acceleration, Reality 1-3, first real fear
- Hour 3: Convergence, all leads to one point
- Hour 4: Finale, Reality 4-6, ending

**Short (3-5 sessions):**
- Session 1: Illusion of normality
- Session 2: First fracture, first escape mechanism
- Session 3: Point of no return, conspiracy Layer 2
- Session 4: Disintegration, Deception Budget maximum
- Session 5: Finale

**Full (6-10 sessions):**
- Sessions 1-2: Surface, establish normalcy
- Sessions 3-4: Cracks, Reality 2, first Time shift
- Sessions 5-6: Immersion, Reality 3, something lost
- Sessions 7-8: Abyss, Reality 4, first Primordial encounter
- Sessions 9-10: Finale, all axes in motion

**Long campaigns don't exist.** Character doesn't survive long.

## ACTION FLOWCHARTS

Step-by-step for common actions. Follow in order.

### Scene Start (before ANY player action)

Run this at the start of every new scene.

```
1. AXES CHECK
   └─ Note current Reality, Time, Identity levels
   └─ Which Narrative Effects apply? (see table)

2. MADNESS CHECK
   └─ 0-2: normal options
   └─ 3-4: prepare 1-2 irrational options
   └─ 5: majority irrational, sane options get DIS

3. DECEPTION BUDGET
   └─ Reality 0: all true
   └─ Reality 1: 1 detail can be false
   └─ Reality 2: 1 NPC/object can be false
   └─ Reality 3: 50% can be false
   └─ Reality 4+: entire scene can be hallucination

4. SET THE SCENE
   └─ Describe with mandatory Narrative Effects
   └─ Add sensory details (use Atmosphere tables if needed)
   └─ Minimum intrusions: Lvl 3 = 1-2, Lvl 4 = 2-3, Lvl 5 = constant

5. PRESENT OPTIONS
   └─ Apply Madness behavioral effects
   └─ Player acts → go to specific flowchart below
```

### Investigation (search, examine, investigate)

```
1. AXIS CHECK
   └─ Reality 2+? → Perception Bleed roll (DC: R2=16, R3=12, R4=8, R5=auto)
   └─ Success? → Player sees "extra" (shadow, symbol, wrong angle)
   └─ Reality 3+? → Layer Slip possible (WIS DC 14 to return)

2. DECEPTION BUDGET
   └─ Reality 0: all true
   └─ Reality 1: 1 detail can be false
   └─ Reality 2: 1 NPC/object can be false
   └─ Reality 3: 50% can be false
   └─ Reality 4+: entire scene can be hallucination

3. MADNESS OPTIONS
   └─ Madness 0-2: normal options
   └─ Madness 3-4: add 1-2 irrational options
   └─ Madness 5: majority irrational, sane options get DIS

4. NARRATIVE STYLE
   └─ Apply Axis Narrative Effects (see table)
   └─ Add intrusions based on highest axis level

5. RESULT
   └─ Present options to player
```

### Social (talk, persuade, interrogate)

```
1. NPC CHECK (in GM thinking)
```

```
   └─ Crit: works, price -1 tier
```
